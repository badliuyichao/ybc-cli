# ybc CLI 项目文档目录

---

## 文档分类说明

本项目文档按照以下两大类进行组织：

### 📁 **design/** - 开发设计文档

**用途**：包含需求设计、架构设计、技术方案、测试策略、安全方案、实施方案等。

**适用场景**：
- 系统设计和架构规划
- 技术选型和方案决策
- API 规范和实施方案
- 测试和安全设计

---

### 📁 **process/** - 开发过程文档

**用途**：包含开发计划、实施报告、验收报告、测试总结、问题追踪等过程性文档。

**适用场景**：
- 开发进度和任务追踪
- 实施过程记录和总结
- 验收和测试报告
- 文档清单和索引

---

### 📁 **根目录** - 用户使用文档

**用途**：包含用户指南、快速上手、常见问题等面向用户的文档。

**适用场景**：
- 快速上手和入门指南
- 详细使用说明
- 常见问题解答

---

**文件命名格式**：
