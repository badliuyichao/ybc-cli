## 🔧 配置方法

### 方法 1：环境变量配置（推荐测试）

**设置新环境变量**（使用新字段名）：
```bash
# Windows CMD
set YBC_TENANT_ID=test-tenant-id-12345
set YBC_APP_KEY=test-app-key-12345678
set YBC_APP_SECRET=test-app-secret-12345678901234
set YBC_ENV=sandbox

# Windows PowerShell
$env:YBC_TENANT_ID="test-tenant-id-12345"
$env:YBC_APP_KEY="test-app-key-12345678"
$env:YBC_APP_SECRET="test-app-secret-12345678901234"
$env:YBC_ENV="sandbox"

# Linux/macOS
export YBC_TENANT_ID="test-tenant-id-12345"
export YBC_APP_KEY="test-app-key-12345678"
export YBC_APP_SECRET="test-app-secret-12345678901234"
export YBC_ENV="sandbox"
```

**向后兼容（旧字段名仍可用）**：
```bash
# 旧环境变量（向后兼容，但新字段优先）
set YBC_AK=test-access-key
set YBC_SK=test-secret-key
```

---

### 方法 2：配置文件配置（推荐持久化）

**⚠️ 注意**：当前 `config init` 命令尚未完全适配新字段，建议先使用环境变量或手动创建配置文件。

**手动创建配置文件** `~/.ybc/config.json`：

**位置**：
- Windows: `C:\Users\<你的用户名>\.ybc\config.json`
- Linux/macOS: `~/.ybc/config.json`

**新格式**（推荐）：
```json
{
  "tenantId": "test-tenant-id-12345",
  "appKey": "test-app-key-12345678",
  "appSecret": "ENCRYPTED_OR_RAW_SECRET",
  "env": "sandbox",
  "format": "table",
  "dataCenter": {
    "gatewayUrl": "https://yonbip.diwork.com/iuap-api-gateway",
    "tokenUrl": "https://yonbip.diwork.com/iuap-api-auth",
    "lastUpdate": "2026-04-27T10:00:00Z"
  },
  "version": "2.0"
}
```

**向后兼容格式**（也支持）：
```json
{
  "tenantId": "test-tenant-id",
  "ak": "test-access-key",
  "sk": "test-secret-key",
  "env": "sandbox"
}
```

**注意**：
- `appSecret` 或 `sk` 会自动加密存储
- 文件权限自动设置为 `600`（仅用户可读写）
- `tenantId` 是必需字段

**症状**：
```
ValidationError: Tenant ID (tenantId) is required
```

**解决方法**：
```bash

# 创建 ~/.ybc/config.json（参考上方格式）
```

---

### 问题 2：签名计算失败

**症状**：
```
AuthError: Failed to calculate signature
```

**调试方法**：
```bash
# 运行签名服务测试验证算法
```

**常见原因**：
- ❌ appSecret 格式错误
- ❌ 时间戳不是毫秒级
- ❌ 参数未排序
